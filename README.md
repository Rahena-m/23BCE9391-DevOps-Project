# VIT-AP Tech Fest

## Project Overview

This project is a static website developed for the DevOps Assignment. It represents the official website for the VIT-AP Tech Fest 2026 event.

The website provides information about:

- Home
- Event Schedule
- Registration
- Contact
- Speakers
